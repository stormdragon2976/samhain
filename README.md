# Samhain

Accessible, simple and fast web client for [Mastodon](https://joinmastodon.org).

Samhain is available at [samhain.social](https://samhain.social). Beta releases are at [beta.samhain.social](https://beta.samhain.social).

For updates and support, follow [@samhain@fosstodon.org](https://fosstodon.org/@samhain).

See the [user guide](https://github.com/NickColley/samhain/blob/main/docs/User-Guide.md) for basic usage. See the [admin guide](https://github.com/NickColley/samhain/blob/main/docs/Admin-Guide.md) if Samhain cannot connect to your instance.
